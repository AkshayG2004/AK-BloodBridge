# AK BloodBridge

## Objective

A web application that connects blood donors and receivers during emergencies by finding nearby eligible donors using geolocation.

## User Roles

1. User
2. Admin

## User Features

- Register
- Login
- Edit Profile
- Update Location
- Enable / Disable Availability
- Search Donors
- Create Blood Request
- Accept Blood Request
- Reject Blood Request
- View Donation History
- View Request History
- Notifications

## Admin Features

- View Users
- View Blood Requests
- Delete Fake Accounts
- Dashboard Analytics
